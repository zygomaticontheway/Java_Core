Задача 2: Управление заказами

Описание задачи:

Создайте программу для управления списком заказов.
Каждый объект "Заказ" должен содержать идентификатор заказа,
имя клиента и сумму заказа.

Программа должна предоставлять функциональность для добавления заказов,
удаления заказов, поиска заказов по имени клиента,
и сортировки заказов по сумме и имени клиента.

Требования:

    1. Создайте класс Order с полями orderID, customerName и amount.
    2. Используйте коллекцию для хранения объектов Order.
    3. Реализуйте методы для добавления, удаления, поиска и сортировки заказов.
    4. Реализуйте сортировку заказов по сумме и имени клиента.
    5. Выведите результаты выполнения каждого действия на экран.
