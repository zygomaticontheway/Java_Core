Задача 1: Каталог книг
Описание задачи:

Вам нужно создать программу для управления каталогом книг.
Каждый объект "Книга" должен содержать название, автора и год издания.
Программа должна предоставлять функциональность для добавления книг,
удаления книг, поиска книг по автору,
и сортировки книг по году издания и названию.

Требования:

    Создайте класс Book с полями title, author и year.
    Используйте коллекцию для хранения объектов Book.
    Реализуйте методы для добавления, удаления, поиска и сортировки книг.
    Реализуйте сортировку книг по году издания и названию.
    Выведите результаты выполнения каждого действия на экран.

Задача 2: Управление заказами

Описание задачи:

Создайте программу для управления списком заказов.
Каждый объект "Заказ" должен содержать идентификатор заказа,
имя клиента и сумму заказа.

Программа должна предоставлять функциональность для добавления заказов,
удаления заказов, поиска заказов по имени клиента,
и сортировки заказов по сумме и имени клиента.

Требования:

    Создайте класс Order с полями orderID, customerName и amount.
    Используйте коллекцию для хранения объектов Order.
    Реализуйте методы для добавления, удаления, поиска и сортировки заказов.
    Реализуйте сортировку заказов по сумме и имени клиента.
    Выведите результаты выполнения каждого действия на экран.
