Написать приложение, которое демонстрирует игру "Лотерея"

1) Игроку выдается 3 билета
2) Каждый билет содержит 5 чисел
3) Игроков может быть несколько
4) Производится розыгрыш, выпадает 5 случайных значений из диапазона от 1 до 36, которые не должны повторяться

--------------------------------------------------------------------------------------------

- билет (Lottery ticket):
    набор значений из 5 элементов в диапазоне от 1 до 36 - массив из 5 элементов

- игрок (Player):
    имя
    набор из 3 билетов - массив

- Лотерея (Lottery):
    количество игроков
    массив для хранения выигрышных номеров
    массив для хранения всех чисел от 1 до 36 - будем его перемешивать
        после перемешивания, возьмем первые 5 элементов
    класс PlayLottery - для метода main и основной логики приложения
